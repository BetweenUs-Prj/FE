컴포넌트 리팩토링 테스트

브라우저에서 http://localhost:5173에 접속하여 다음을 확인해주세요:

✅ 확인 사항:

  • 콘솔 오류 없음: F12 → Console 탭에서 import 오류가 없어야 함
  • 페이지 로딩: 메인 페이지가 정상 로딩되어야 함
  • 게임 메뉴: 퀴즈/반응속도 게임 버튼이 보여야 함
  • 네비게이션: 게임 선택 화면으로 이동 가능해야 함
